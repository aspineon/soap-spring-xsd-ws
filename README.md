soap-spring-xsd-ws
==================

This is simple Web service implementation using SOAP and Spring with XSD file for service document object generation using JAXB.


<h4>Technogies</h4> 
<hr/>
<ol>
<li> JAVA 1.6</li>
<li> SOAP </li>
<li> Spring 3.x </li>
<li> Jersey / REST</li>
<li> XSD </li>
<li> JAXB 2 </li>
<li> Maven </li>
<li> SOAP UI </li>
</ol>
